## 🚀 Getting Started

### Prerequisites

- A modern web browser.
- Git.
- Optional: Python `http.server` or VS Code Live Server extension to run locally with hot-reloading.

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ayonbiswas567/Portfolio-Application-.git
   cd Portfolio-Application-
   ```

2. Start a local server:
   - Using Python:
     ```bash
     python -m http.server 8000
     ```
   - Or open `index.html` using VS Code's **Live Server** extension.

3. Open your browser and navigate to `http://localhost:8000`.

## 🌐 Deployment on Netlify

This website is ready for zero-config deployment on Netlify:

1. Drag-and-drop the project folder to the **Netlify App dashboard**, or connect this GitHub repository directly to Netlify.
2. Netlify will host the static `index.html`, `style.css`, and `script.js` instantly.
3. Ensure that your remote API key is active on GitHub if you are using the native chatbot system.
